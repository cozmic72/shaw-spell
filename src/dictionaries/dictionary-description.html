The Shavian alphabet (also known as the Shaw alphabet) is a phonetic alphabet for the English language.

This dictionary, released by joro.io, is part of Shaw-Spell $VERSION$, providing comprehensive Shavian-English translation and definition support for macOS.

Compiled using Shavian.info's ReadLex dictionary for translations and IPA transcriptions, and Open English WordNet 2024 for the word descriptions.
