About Shaw-Spell Dictionary

The Shavian alphabet (also known as the Shaw alphabet) is a phonetic alphabet for the English language.

This web dictionary, released by joro.io, is part of Shaw-Spell $VERSION$, providing comprehensive Shavian-English translation and definition support.

Compiled using Shavian.info's ReadLex dictionary for translations and IPA transcriptions, and Open English WordNet 2024 for the word descriptions.

Features

  • Search in both Latin and Shavian scripts
  • English-to-Shavian and Shavian-to-English dictionaries
  • Support for both GB and US dialects
  • Optional Shavian-only definitions for immersive learning
