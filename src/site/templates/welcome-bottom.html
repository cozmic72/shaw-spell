About the project

TShaw-Spell is a bidirectional Shavian-"English" dictionary that I have developed with Claude Code. Also available as a native macOS dictionary and spell-checker!

The dictionaries were built on top of Readlex, the authoritative source of truth for Shavian spelling. The word definitions come from Open Wordnet. The macoOS spelling-correction engine is built using Hunspell, again using the Readlex and Wordnet dictionaries.
